# Rombadil

Rombadil is a cycle accurate NES emulator implemented in C# using OpenTK (OpenGL, OpenAL, GLFW).

## Utils

- `rombadilasm` is a 6502 assembler
- `rombadil6502` is a standalone 6502 CPU emulator
